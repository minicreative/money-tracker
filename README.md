# money-tracker

### Setup

Do `npm install`  
Create `./config.json` with the following format:
```
{
	"port": 3000,
	"database": {
		"host": "127.0.0.1",
		"name": "money-tracker",
		"username": "user",
		"password": "password"
	}
}
```

### Run
`npm run dev` to watch serve and watch UI and API in paralell  
`npm run start` to build application and start server demon
